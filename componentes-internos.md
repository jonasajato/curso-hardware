<h4><strong>FONTE</h4></strong>
-> Fornece energia elétrica aos componentes do computador;<br>
-> A função da fonte é receber a tensão fornecida pela rede elétrica (110 ou 220 volts) e converter essa tensão em uma voltagem que atenda as 
necessidades de cada dispositivo do computador, ou seja, em 3,3V, 5V ou 12V;<br>
-> Atenua os picos de energia para evitar queima dos componentes;<br>
-> Escolha da fonte: potência maior que a soma do consumo de energia dos componentes internos e externos;<br>
-> A fonte é um dos itens com maior probabilidade de falha, portanto, cuidados devem ser tomados quando instalarmos e 
utilizarmos uma fonte de alimentação do computador, como limpá-la sempre que necessário, por exemplo.<br>
<br>
<a id="ancora2"></a>
<h4><strong>PLACA MÃE</h4></strong>
-> Responsável por ligar e conectar todos os componentes;<br>
-> Tipos de placa mãe:<br>
--> AT (Advanced Technology, seu uso foi constante de 1983 até 1996);<br>
--> ATX (Advanced Technology Extended, usadas atualmente);<br>
--> BABY AT (versão menor da AT original);<br>
--> BTX (criado pela Intel e lançado em 2003 para substituir o formato ATX);<br>
--> ITX (criado em outubro de 2001 pela VIA Technologies);<br>
--> LPX (criada para permitir PCs mais finos);<br>
--> NLX (criada para microcomputadores que usam processadores Pentium III e 4).<br>
-> Controladores On-board e Off-board:<br>
--> On-board = já está na placa, exemplos: entradas usb, conector vga;<br>
--> Off-board = será conectado na placa, exemplos: placa de vídeo, placa de rede;<br>
-> Componentes e Conectores:<br>
--> Processador (conectado ao soquete)<br>
--> Memória RAM<br>
--> Bios (memória ROM)<br>
--> Bateria<br>
--> Chipset (norte e sul)<br>
--> Slots de expansão (PCI, ISA, AGP...)<br>
--> Conector IDE<br>
--> Conector SATA<br>
--> Conector Mouse<br>
--> Conector Teclado<br>
--> Conector Impressora (porta paralela)<br>
--> Conector USB<br>
-> Barramentos:<br>
--> Os barramentos são vias de comunicação entre os diversos componentes da placa mãe;<br>
-> Barramento Local:<br>
--> Também chamado de FSB (Front Side Bus), consiste em um barramento de alta velocidade que trabalha na mesma frequência do processador;<br>
-> Barramento de Expansão:<br>
--> São os barramentos responsáveis por possibilitar a conexão de placas de expansão no computador, tais como: Placa de vídeo, Placa de Som, Placa de rede, etc.<br>
-> Existe uma grande variedade de barramentos. Os principais são:<br>
--> ISA (Industry Standard Architecture)<br>
--> EISA (Extend Industry Standard Architecture)<br>
--> VLB (Vesa Local Bus – VESA)<br>
--> AMR (Audio and Modem Riser) e CNR (Communications and network Riser)<br>
--> PCI (Peripherical Component Interconect)<br>
--> AGP (Accelerated Graphics Port)<br>
--> PCI – Express<br>
--> USB – Universal Serial Bus<br>
--> Firewire (IEEE 1394)<br>
--> HDMI<br>

<h4><strong>BIOS</h4></strong>
-> É um firmware:<br>
--> O firmware é o conjunto de instruções operacionais programadas diretamente no hardware de um equipamento eletrônico. 
É ele que contém as informações de inicialização que permitem o correto funcionamento do aparelho;<br>
--> Ele é responsável pelo suporte básico de acesso ao hardware, bem como por iniciar a carga do sistema operacional. 
A BIOS fica gravada em uma memória ROM, impedindo-a de ser desinstalada.<br>

<h4><strong>PROCESSADOR</h4></strong>
-> O processador, Microprocessador ou CPU (Central Processing Unit – Unidade Central de Processamento), é o principal componente do computador;<br>
-> Atualmente existem apenas três empresas especializadas na fabricação deste componente (Intel, AMD e VIA);<br>
-> <strong>O processador é um chip composto por milhões de transistores (Transistores correspondem a unidade básica do processador, 
  capaz de processar 1 bit de cada vez.</strong><br>
  
<br>
<a id="ancora5"></a>
<h4><strong>COOLER</h4></strong>
-> Os processadores são dispositivos que emitem grande quantidade de calor, havendo, portanto, a necessidade de um sistema de resfriamento eficiente, 
para evitar a queima do componente; <br>
-> E é o Cooler o responsável pela dissipação do calor emitido pelo processador.<br>
-> Este dispositivo exige também alguns outros cuidados para evitar problemas com o computador e aumentar sua vida útil:<br>
--> Verificar se o cabo de energia do cooler está conectado a placa-mãe quando montar o computador, pois, caso isso aconteça, não haverá refrigeração do processador, podendo queimar o mesmo. 
--> Verificar se o Cooler está encaixado e firme ao soquete;<br>
--> Sempre realizar uma limpeza devido ao acumulo de poeira, que pode aumentar a temperatura, vindo a prejudicar o processador.<br>

<br>
<a id="ancora6"></a>
<h4><strong>PASTA TÉRMICA</h4></strong>
-> Empregada entre o dissipador de calor do cooler e o processador, ela é um dos elementos responsáveis por evitar o superaquecimento de temperatura do computador;<br>
-> Considerando-se que as superfícies tanto do processador quanto do cooler não são completamente lisas, surgem entre eles microdepressões;<br>
-> Essas microdepressões microscópicas são capazes de formar espaços entre os dois componentes. Os espaços vazios formam ar;<br>
-> O objetivo da pasta térmica é justamente preencher esses espaços, por se entender que o ar não é um bom condutor de calor;<br>
-> Aplicada corretamente sobre o processador, a pasta térmica ocupa essas micro depressões, permitindo uma refrigeração mais eficiente do processador.<br>

<h4><strong>MEMÓRIA</h4></strong>
-> Chamamos memória a todos os dispositivos com capacidade de armazenamento de dados. 
Podemos dividir o armazenamento em duas categorias: Armazenamento Temporário e Armazenamento Secundário;<br>
--> <strong>Armazenamento Temporário:</strong> São dispositivos que armazenam determinados dados enquanto está sendo fornecida energia ao mesmo, ou seja, 
quando desligamos o computador, estes dados são perdidos. 
Também são chamados de Memória Volátil. Os tipos de memória pertencentes a esta categoria são RAM (Randomic Access Memory – Memória de Acesso Aleatório) e 
ROM (Read Only Memory – Memória Somente Leitura);<br>
--> <strong>Armazenamento Secundário:</strong> São dispositivos que permitem o armazenamento permanente, ou seja, os dados só serão apagados 
caso o usuário ou o sistema efetue a exclusão. Nesta categoria podemos citar o Disco Rígido, Pen-Drive, CD, DVD, etc.<br>

<br>
<a id="ancora8"></a>
<h4><strong>MEMÓRIA ROM</h4></strong>
--> <strong>PROM (Programmable ROM)</strong> – Memória fabricada sem programação de fábrica, sendo possível programá-la apenas uma vez. Atualmente não é mais utilizada;<br>
--> <strong>EPROM (Eraseable PROM)</strong> – Tipo de memória ROM que possibilita o apagamento e reprogramação através de uma “janela de vidro” onde raios ultravioleta de alta potência incidem fazendo com que os dados sejam apagados;<br>
--> <strong>EEPROM (Electrically Eraseable PROM)</strong> – Esse é o tipo de memória ROM usada atualmente. Permite sua reprogramação através da utilização de disquetes, ou então, de Hardware instalado no computador. A este procedimento dá-se o nome de “atualização de BIOS”.<br>

<br>
<a id="ancora9"></a>
<h4><strong>MEMÓRIA RAM</h4></strong>
--> A memória RAM é a memória principal do computador e é o dispositivo que permite o armazenamento temporário de dados, por esse motivo também é chamada de Memória Volátil, ou seja, os dados são apagados no caso de ausência de energia elétrica, por exemplo, quando o computador é desligado;
<br>
--> O processador não possui capacidade de armazenamento, então, é na RAM que ele busca os dados (operação de “leitura”) ou armazena os dados (operação de “escrita”), por exemplo, quando você executa um determinado programa, ele é transferido do HD para a memória RAM, onde o processador irá ler os dados e executá-lo. Portanto, quanto mais memória o seu computador tiver, mais programas poderão rodar simultaneamente.
